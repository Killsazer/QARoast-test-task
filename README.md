# Quiz Builder

A full-stack quiz creation platform built with **Nest.js** (backend) and **Next.js** (frontend).

## Tech Stack

| Layer    | Technology                                      |
| -------- | ----------------------------------------------- |
| Backend  | Nest.js, TypeScript, Prisma, PostgreSQL          |
| Frontend | Next.js (App Router), React, Tailwind CSS, Zod   |
| Database | PostgreSQL 16 (via Docker)                       |

## Prerequisites

- [Node.js](https://nodejs.org/) >= 18
- [Docker](https://www.docker.com/) & Docker Compose
- npm or yarn
